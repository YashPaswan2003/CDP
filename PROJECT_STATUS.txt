7-AGENT PIPELINE (LangGraph):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. DataIngestionAgent
   Input:  CSV file (Google Ads / DV360 / Meta)
   Output: Normalized rows in PostgreSQL

2. ClientAnalysisAgent
   Input:  client_id + date_range
   Output: KPI summary JSON + trend flags

3. OrchestratorAgent
   Input:  client_id
   Output: Triggers all agents, master sheet

4. VisualizationAgent
   Input:  metrics query
   Output: ECharts/Recharts config JSON

5. CDPGodAgent
   Input:  natural language question
   Output: Accurate answer + citations (omniscient knowledge base)

6. ChatbotAgent
   Input:  user message + session context
   Output: Streaming response (uses CDPGodAgent)

7. PresentationAgent
   Input:  client_id + template_id
   Output: .pptx file in Cloudflare R2
